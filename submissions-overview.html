Resource Submission Process

With the ABC Hub (the Hub) we are collectively building a platform to connect people and places with ideas and actions. While the resources and community hosted here are open to all, what makes the Hub an engine for change is active participation. Participation can take many forms. Most essential to the growth of the Hub are three different ways to contribute directly to the content of the Hub.
  • Add a profile page of your self and/or your organization. This step is required before you submit a resource or blog post.
  • Add a resource page the provides the Hub community with tools and techniques for enacting change.
  • Add a blog post that generate fruitful dialogue or brings awareness to a particular event or action.
This page provides you with the steps and resources you need to submit this content for inclusion on the the Hub website. To submit your content, you will need a computer/tablet/phone with internet access as a simple text editor. Below we wll provide you with a step by step guide that uses Zettlr, our recommended Free and Open Source Software for document editing. Following these steps will simplify and expedite your submission process and also ensure that your content renders in a predicable way on the Hub website.

Step 1: Downloading, installing, and learning Zettlr.

  1. Download and install a Markdown editor. We recommend Zettlr if this is your first time with Markdown. For step by step instruction to get up and running with Zettlr. Go to our Markdown Instructions Page
  2. Download our submission templates by clicking the link below and downloading the folders and files from our Google Drive. (No Google account required)
  3. Prepare your individual or organization community profile by following the steps on the Community Profile Submission page.
  4. Submit your profile via one of two available paths: Github (recommended, GitHub account required) or link to a Google Drive folder (Google account required)

  2. Based on you operating system (MAC, Windows, Linux, etc), select the appropriate button below the respective icons to install the installer package.
  3. As the download begins, a pop-up window tilted "Next steps" should appear with suggestions and links for learning how (and why) to use Zettlr. There is also a series of embedded videos. We highly recommend that you take some time to review and bookmark this material.
  4. Navigate to your downloads folder and double click on the Zettlr download package to begin the installation set-up prompts. Default setting will be appropriate for most users.
  5. Be sure that the "Run Zettlr" box is checked when click "Finish" on the final intallation window, and the Zettlr application will open automatically.
  6. To further aid your learning how to use Zettlr, the installation includes a folder that contains several example files that include more information on advanced functionality in Zettlr. You do not need to know these advanced features, however, we recommned you this preloaded folder (what Zettlr calls a workspace) to become familiar with the user inferace and some of the methods for formatting.
