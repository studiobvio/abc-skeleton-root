Community Profile Submission Form

Fill out the fields below to generate and format the metadata to be copied and pasted at the the top of your submission markdown file. Download the template markdown files and directory with the button below.

Basic Info

If creating a profile for an organization of community, add the full name (with not special characters) into the "First Name" input and leave "Last Name" input blank. Asterisks signify required fields. The "Unique ID" should be an easy to remember two digit number that will be appended to your name to form your unique profile name.
