Resource Submission Process

To make resources in the ABC Repository as widely accessible and extensible as possible to our community, we require all resources conform to the strict formatting guidelines outline below. For each resource you will upload a single folder with a primary text file and any supplemental media files in ato our open-source repository which is hosted on GitHub. The repository content will be rendered as a searchable website. While you may draft your resource material an any format you wish, your final submission will be a MarkDown document (learn more about Markdown here).
